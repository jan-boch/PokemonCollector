This is a Pokemon Tracker project

## Getting Started

Always install dependencies with
```npm install --save-optional --os=linux --cpu=x64```
to make sure dependencies ally with vercel linux deployment

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [https://pokemon-collector-six.vercel.app/](http://localhost:3000) with your browser to see the result.
